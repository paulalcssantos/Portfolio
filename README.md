### 🇧🇷 Português

Bem-vindo ao repositório do meu portfólio profissional. Este site foi desenvolvido para centralizar minha trajetória técnica em dados, minhas contribuições em comunidades globais de tecnologia e minha atuação como coautora e palestrante.

#### 🎨 Design & Estética
O projeto utiliza uma paleta **"Autumn Dark"** (Tons de outono), focada em transmitir maturidade, elegância e autoridade técnica. 

#### 🛠️ Tecnologias Utilizadas
- **HTML5 & Tailwind CSS**: Estrutura e estilização responsiva de alta performance.
- **Vanilla JavaScript**: Lógica de tradução (PT/EN), manipulação dinâmica de conteúdo e menu mobile.
- **Font Awesome**: Ícones sociais e técnicos.
- **GitHub Pages**: Hospedagem gratuita e integrada.

#### ✨ Destaques do Projeto
- **Sistema Bilíngue**: Tradução completa via objetos JSON no script, sem dependência de APIs externas de tradução.
- **Trajetória Dinâmica**: Divisão por abas para experiências Profissionais, Voluntárias e Internacionais.
- **Mural de Evidências**: Seção de aulas com vídeos embutidos e prova social via feedbacks.
- **Responsividade Total**: Experiência otimizada para Desktop, Tablets e Mobile (Menu Hambúrguer).

### 🇺🇸 English

Welcome to my professional portfolio repository. This website was built to centralize my technical journey in data, my contributions to global tech communities, and my work as a co-author and keynote speaker.

#### 🎨 Design & Aesthetics
The project follows an **"Autumn Dark"** palette, curated to convey maturity, elegance, and technical authority.

#### 🛠️ Tech Stack
- **HTML5 & Tailwind CSS**: High-performance responsive structure and styling.
- **Vanilla JavaScript**: Translation logic (PT/EN), dynamic content rendering, and mobile menu handling.
- **Font Awesome**: Technical and social iconography.
- **GitHub Pages**: Fast and integrated hosting.

#### ✨ Project Highlights
- **Bilingual System**: Full translation handled via JSON objects in JS, ensuring technical accuracy in both languages.
- **Dynamic Trajectory**: Tabbed navigation for Professional, Voluntary, and International experiences.
- **Evidence Mural**: Dedicated classes section with embedded videos and social proof through student feedback.
- **Fully Responsive**: Optimized experience for Desktop, Tablet, and Mobile devices (Hamburger menu).
